jQuery

Bootstrap

Bootstrap是一组用于制作网站及网络应用程序的工具，里面包含有HTML、层叠样式表（CSS）及JavaScript的框架，提供字体排印、表单、按钮、导航及其他各种元件，并提供了Javascript扩充套件。对于同时要供不同上网装置及浏览器，如一般电脑、手机及平板电脑等浏览的网站及网络应用程序，Bootstrap提供了CSS媒体查询（Media Query），节省网站制作者分别制作不同版本的时间及不便。

据2014年6月的统计，Bootstrap在GitHub里面排名第一，获得了7万3千颗星的评价。不少热门网站如MSNBC与美国航太总署（NASA）等均使用了Bootstrap。
